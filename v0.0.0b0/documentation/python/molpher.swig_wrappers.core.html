1.1.1.2.1.1. molpher.swig_wrappers.core module¶

A collection of functions and classes that expose the C++ API to Python.

Note

This interface is automatically generated by SWIG and tightly follows the implementation in C++.

Attention

The classes and methods in this module are not intended for direct use and should not be imported by external scripts. The molpher.core module contains classes that are more convenient and allow a more Pythonic way of doing things.
